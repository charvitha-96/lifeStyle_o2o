### User Authentication & Registration
**Description:**
Implement a secure user authentication and registration system allowing users to sign up, log in, and reset their passwords.

**Acceptance Criteria:**
- Users can sign up with valid email and password.
- Users receive a confirmation email upon signing up.
- Users can log in with their credentials.
- Users can reset their password via email verification.

**Labels:**
feature, user-authentication, high-priority
